# Boilerplate litElement + PUG Template + TypeScript + JSON Data

- Docker
- Gulp
- Webpack
- Php
- Pug templates
- Sass
- JSON files for the data
- TypeScript
- LitElement

## Start the project (first time)

### First time

```shell
chmod -R +x bin

make coffee
```

### Development mode

#### Stop the container

```shell
make start
```

#### Start the container

```shell
make start
```

#### Development

```shell
make build

make watch
```

or just

```shell
make bw
```
